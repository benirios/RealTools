|-----------|----------|-----------|-------------|-----------------|

| T-18-02 | Information Disclosure | `StrategySelector` | accept | Selector contains public strategy labels only; no user score data crosses this component. |
| T-18-03 | Tampering | score values | mitigate | Selector emits only a strategy slug and never accepts or computes score values. |
| T-18-04 | Information Disclosure | raw scoring JSON | accept | Selector does not render scoring JSON or persisted rows. |
